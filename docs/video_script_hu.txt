6. VIDEOHOZ HASZNALHATO ROVID LEIRAS

Title:
Real2Sim G1 - Camera-based two-arm imitation in MuJoCo

Description:
This project demonstrates a Python real2sim MVP for the SZTAKI Robot Competition 2026 task. A camera observes human motion, MediaPipe extracts pose landmarks, an approximate retargeting module maps human arm motion to a Unitree G1 MuJoCo model, and an OpenClaw-compatible command wrapper exposes demo commands. The stable submission mode imitates both human arms. An experimental stereo full-body extension demonstrates torso, squat, and leg motion.
